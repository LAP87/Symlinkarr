# Anime Cleanup

Use this page when legacy anime layouts, duplicate roots, or blocked anime cleanup groups still need human review.

## What This Is For

Anime cleanup exists because older Plex/Hama-style or mixed-root layouts can leave the filesystem in a state that normal scan and cleanup should not silently rewrite.

The goal is:

- identify legacy groups
- explain why they are blocked
- produce a reviewable plan before changing files

## When You Actually Need It

You probably need anime cleanup when:

- anime content exists in both tagged and untagged roots
- Hama-era layouts still exist
- cleanup keeps reporting anime-specific blocked groups
- duplicate anime root structures are surfacing in reports

## What It Is Not

It is not a general-purpose day-to-day dashboard.

Most people should only touch it when normal cleanup or scan pages point them there.
